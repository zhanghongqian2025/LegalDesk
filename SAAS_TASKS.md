# LegalDesk SaaS 版开发任务

## 项目概述

- **项目名称**: LegalDesk SaaS
- **定位**: 面向个人的 AI 法律咨询 Web 应用
- **核心优势**: 即时使用（无需下载安装）
- **目标用户**: 普通大众（非律师）

## 产品规格

| 项目 | 内容 |
|------|------|
| **形态** | Web 应用 |
| **核心服务** | 法律咨询 + 诉讼指导 |
| **价格** | 免费 |
| **数据存储** | 完全云端 |

## 功能需求

### Phase 1: MVP

- [ ] 用户注册/登录系统
- [ ] 法律咨询对话（AI 问答）
- [ ] 诉讼指导（起诉状生成）
- [ ] 案件管理（创建/查看/删除）
- [ ] 云端数据存储

### Phase 2: 扩展功能

- [ ] 合同审核

- [ ] 证据分析
- [ ] 案例检索
