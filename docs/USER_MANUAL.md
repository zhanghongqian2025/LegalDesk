## 2. 上传文件

### 2.1 功能入口
应用启动后默认显示「上传文件」页面。

### 2.2 操作步骤

1. **点击上传区域** 或直接拖拽文件到虚线框内
2. 选择需要上传的法律文件
3. 支持的文件格式：
   - PDF 文档
   - Word 文档（.doc, .docx）
   - 图片（.jpg, .jpeg, .png）
   - 文本文件（.txt）

### 2.3 AI 自动分析

文件上传后，系统会自动：
- 读取文件名提取关键信息
- 生成案件名称建议
- 判断案件类型
- 提供案件描述建议

### 2.4 后续操作

AI 分析完成后，你可以：
- **创建案件**：确认 AI 建议的信息，创建新案件
- **选择已有案件**：将文件关联到现有案件

## 3. 案件管理

### 3.1 查看案件列表

1. 点击侧边栏「案件管理」
2. 显示所有案件的卡片列表
3. 支持搜索和筛选：
   - 按案件名称或案号搜索
   - 按案件类型筛选（民事/刑事/行政/仲裁/非诉）
   - 按案件状态筛选（待处理/办理中/已结案/已归档）

### 3.2 创建新案件

1. 点击「新建案件」按钮
2. 填写案件信息：
   - **案件名称**（必填）：如「张某诉李某合同纠纷」
   - **案号**：如「(2024) 民初字第1234号」
   - **案件类型**：民事/刑事/行政/仲裁/非诉
   - **案件状态**：待处理/办理中/已结案/已归档
   - **法院**：受理法院名称
   - **对方当事人**：对方名称
   - **立案日期**：案件受理日期
   - **开庭日期**：预计开庭时间
   - **案件描述**：案件基本情况

### 3.3 编辑案件

点击案件卡片进入详情页，点击「编辑」按钮修改案件信息。

### 3.4 删除案件

在案件列表中，点击删除图标，确认后删除案件（不可恢复）。

## 4. 卷宗整理

### 4.1 功能入口
进入案件详情后，默认为「卷宗材料」标签页。

### 4.2 上传文件

1. 点击「上传文件」按钮
2. 选择本地文件
3. 文件自动添加到案件卷宗

### 4.3 管理文件

- **查看**：显示文件名和类型
- **删除**：点击删除图标移除文件

### 4.4 文件分类

卷宗文件分为四类：
- 证据材料
- 法律文书
- 庭审记录
- 往来函件

## 5. 法律文书

### 5.1 功能入口
在案件详情页点击「法律文书」标签。

### 5.2 创建文书

1. 点击「新建文书」按钮
2. 填写文书信息：
   - **文书标题**：如「民事起诉状」
   - **文书类型**：起诉状/答辩状/代理词/辩护词/法律意见书/合同等
   - **文书内容**：文书正文内容

### 5.3 编辑文书

点击已有文书卡片，修改内容后保存。

### 5.4 删除文书

点击删除图标，确认后删除文书。

### 5.5 版本管理

系统自动记录文书版本号（v1, v2, ...）。

## 6. 证据审核

### 6.1 功能入口
在案件详情页点击「证据审核」标签。

### 6.2 添加证据

1. 点击「添加证据」按钮
2. 填写证据信息：
   - **证据名称**：如「合同原件」
   - **证据类型**：物证/书证/证人证言/间接证据
   - **真实性**：已核实/未核实/有争议
   - **合法性**：合法/非法/存疑
   - **关联性**：有关联/无关联/存疑
   - **证据分析**：详细分析说明

### 6.3 证据三性分析

系统提供证据三性分析：
- **真实性**：证据是否真实可靠
- **合法性**：证据取得方式是否合法
- **关联性**：证据与案件事实是否相关

颜色标识：
- 🟢 绿色：已核实/合法/有关联
- 🟡 黄色：未核实/存疑
- 🔴 红色：有争议/非法/无关联

### 6.4 证据分析

在「证据分析」字段可以详细记录：
- 证据的来源
- 证明内容
- 采信理由
- 争议点

## 7. AI 助手

### 7.1 功能入口
在案件详情页点击「AI 助手」标签。

### 7.2 功能特点

AI 助手可以帮你：
- 📝 **起草法律文书**：根据案件信息生成文书初稿
- 🔍 **分析证据三性**：智能分析证据的真实性、合法性、关联性
- 📁 **整理卷宗材料**：对上传文件进行分类整理
- ⚖️ **法律咨询解答**：解答法律问题

### 7.3 使用方法

1. 在输入框中输入你的问题或需求
2. 按 Enter 或点击发送按钮
3. AI 会根据当前案件信息给出回复

### 7.4 对话示例

```
用户：帮我起草一份起诉状
AI：好的，请提供以下信息：
- 原告和被告基本信息
- 诉讼请求
- 事实与理由

用户：这是一个合同纠纷案件...
AI：（根据案件信息生成起诉状草稿）
```

### 7.5 配置 API

在使用 AI 功能前，需要先配置 API：
1. 前往「设置」
2. 输入 API Key
3. 填写 API 地址
4. 保存配置

## 8. 模板管理

### 8.1 功能入口
点击侧边栏「文书模板」。

### 8.2 创建模板

1. 点击「新建模板」按钮
2. 填写模板信息：
   - **模板名称**：如「民事起诉状模板」
   - **分类**：如「起诉状」「合同」
   - **变量**：用逗号分隔，如「原告,被告,案由,诉讼请求」
   - **模板内容**：文书内容，使用 `{变量名}` 表示需要替换的部分

### 8.3 模板示例

```
名称：民事起诉状
变量：原告,被告,案由,诉讼请求

内容：
民事起诉状

原告：{原告}
被告：{被告}

案由：{案由}

诉讼请求：
{诉讼请求}

此致
XX人民法院

具状人（签名）：
年  月  日
```

### 8.4 使用模板

在创建法律文书时，可以选择使用已保存的模板。

## 10. 常见问题

### Q1: 如何备份数据？
前往「设置」-「数据管理」-「导出数据」

### Q2: AI 功能需要付费吗？
AI 功能需要配置 API Key，费用由你使用的 AI 服务商收取。

### Q3: 数据存储在哪里？
所有数据存储在本地设备，不会上传到云端，确保数据安全。

### Q4: 支持哪些文件格式？
支持 PDF、Word（.doc/.docx）、图片（.jpg/.jpeg/.png）、文本文件（.txt）。

### Q5: 如何联系技术支持？
如有问题，请联系技术支持团队。
