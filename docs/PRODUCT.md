# LegalDesk 产品规格说明书

## 1. 产品概述

### 1.1 产品定义
**LegalDesk** 是一款专为法律工作者设计的本地桌面 AI 助手应用，旨在帮助律师、法务、法官、检察官等法律从业者高效管理案件、整理卷宗、起草法律文书、审核证据。

### 1.2 产品定位
- **目标用户**：律师、法务人员、法律顾问、法官、检察官、法律学者
- **使用场景**：日常办公、案件管理、文书起草
- **核心价值**：提升法律工作效率，智能化辅助法律工作

### 1.3 产品特点
- 🖥️ **本地桌面应用** - 数据存储在本地，保护隐私安全
- 🤖 **AI 智能辅助** - 自动分析文件，智能生成文书
- 📁 **全流程管理** - 从文件上传到案件归档全覆盖
- 🎨 **简洁易用** - 友好的用户界面，流畅的操作体验

### 2.2 详细功能说明

#### 2.2.1 案件管理
- 支持创建民事、刑事、行政、仲裁、非诉五类案件
- 案件状态：待处理、办理中、已结案、已归档
- 支持案号、法院、当事人等案件信息管理
- 支持立案日期、开庭日期、期限等时间管理
- 案件搜索和筛选功能

#### 2.2.2 卷宗整理
- 支持 PDF、Word、图片、文本等多种格式
- 文件分类管理：证据材料、法律文书、庭审记录、往来函件
- 快速上传和删除功能

#### 2.2.3 法律文书
- 支持多种文书类型：起诉状、答辩状、代理词、辩护词、法律意见书、合同等
- 富文本编辑
- 版本历史记录
- 支持自定义模板

#### 2.2.4 证据审核
- 证据类型：物证、书证、证人证言、间接证据
- 证据三性分析：
  - 真实性：已核实/未核实/有争议
  - 合法性：合法/非法/存疑
  - 关联性：有关联/无关联/存疑
- 详细的证据分析记录

#### 2.2.5 AI 助手
- 基于当前案件信息的智能对话
- 辅助起草法律文书
- 证据三性分析
- 法律问题解答
- *需配置 API Key*

#### 2.2.6 模板管理
- 自定义文书模板
- 变量支持（{变量名} 语法）
- 模板分类管理

## 3. 用户界面设计

### 3.1 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  LegalDesk                    [侧边栏]    │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                 │
│  📤 上传 │           主要工作区                             │
│          │                                                 │
│  📁 案件 │    - 上传文件页面                               │
│          │    - 案件列表页面                               │
│  📄 模板 │    - 案件详情页面                               │
│          │    - 模板管理页面                               │
│  ⚙️ 设置 │    - 设置页面                                   │
│          │                                                 │
└──────────┴──────────────────────────────────────────────────┘
```

### 3.2 设计原则

1. **简洁优先** - 减少视觉干扰，聚焦核心内容
2. **一致性** - 统一的视觉风格和交互模式
3. **可访问性** - 清晰的层次结构，易于导航
4. **反馈及时** - 操作结果即时可见

### 3.3 视觉风格

- **配色方案**：蓝色为主色调，专业稳重
- **字体**：系统字体，清晰易读
- **间距**：适度留白，舒适阅读
- **圆角**：12px 圆角，柔和视觉
- **阴影**：轻微投影，增加层次感
